# Youtube-Video-Downloader

```
Run "Youtube Video Downloader.exe" on Windows to run this app
```

If python >= 3.8.5 is installed on the machine:

> Windows:
```
$ pip install -r requirements.txt
$ python app.py
```

> Mac / Linux:
```
$ pip3 install -r requirements.txt
$ python3 app.py
```
